# SnoozeSavvy Pin Scheduler

SnoozeSavvy Pin Scheduler is an internal publishing workflow for SnoozeSavvy.com.

The app helps the SnoozeSavvy team prepare and publish approved organic Pins for its own sleep-related articles, tools, and guides.

## Purpose

This project is designed to reduce manual publishing time for SnoozeSavvy’s own Pinterest content calendar.

It supports:

- approved Pin image previews
- Pin title, description, destination URL, and alt text
- target board selection
- Pinterest OAuth connection
- preparation for publishing approved Pins through the official Pinterest API

## Compliance

This app is not a public automation service.

It does not:

- scrape Pinterest
- repost third-party content
- automate other users’ accounts
- publish spam
- bypass Pinterest controls

All Pin content is created, reviewed, and approved by the SnoozeSavvy team before publishing.
